CONOCIMIENTOS

  • Figma

    Figma es una aplicación para hacer UI. Su principal objetivo es desarrollar interfaces de usuario dentro del navegador, sin tener que acudir a software nativo.

  • Marvel

    Marvel es una plataforma de diseño colaborativo. Wireframe, prototipo, diseño en línea y crea especificaciones de diseño en un solo lugar.

  • Maze

    Maze es una plataforma de prueba para el usuario que convierte su prototipo en información procesable de usuarios reales, brindando confianza al proceso de diseño.

  • Zeplin

    Zeplin es un software que permite incorporar los diseños cogiendo la guía de estilo, y que permite exportar los iconos de forma automatizada.

  • Github

    GitHub es un servicio de alojamiento que ofrece a los desarrolladores repositorios de software usando el sistema de control de versiones, Git. Crea código fuente.

  • HTML5

    HTML5 es el lenguaje con el que se define el contenido de las páginas web. Es un conjunto de etiquetas que definen el texto y otros elementos que compondrán una página web.

  • CSS3

    CSS3 es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​

  • JavaScript

    JavaScript es el lenguaje de programación más importante y potente, permite a los desarrolladores crear acciones en sus páginas web.
